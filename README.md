# 💹 日元薪资转换器 (JPY Salary Converter)

一个极简、支持 PWA 离线使用的日元薪资换算工具。帮助在日华人快速折算月薪、年薪及汇率购买力。

## 🚀 核心功能
* **实时汇率**：自动同步最新汇率，支持一键重置。
* **税前/税后**：支持税前额面输入，自动估算 20% 扣除额。
* **多维换算**：自动计算年薪、周 40 小时折算时薪。
* **汇率模拟**：3.5 - 5.0 汇率区间对比，内置可视化图表。
* **离线使用**：支持 PWA，可安装至手机桌面并离线运行。

## 🛠 快速开始
1. 克隆代码：
   ```bash
   git clone [https://github.com/ISunki/jpy-calculator.git](https://github.com/ISunki/jpy-calculator.git)
   ```
2. 浏览器打开 `index.html` 即可使用。
